# 📚 StudyTrack

A personal study chapter tracker to manage your learning progress.

## 🌐 Live Site
[Open App](https://study-tracker-two-weld.vercel.app)

## ✨ Features
- Add subjects with custom colors
- Add chapters per subject
- Track 3 learning steps per chapter:
  - 🎬 Watching Class
  - 📖 Self Study
  - 🔄 Revision
- Each step: ○ empty · ✓ done · ✗ skipped
- Progress bar per subject
- Data saves automatically

## 📱 Add to Home Screen
Open in Chrome → ⋮ menu → Add to Home Screen
